Description

CoderSpace creates engaging learning experiences along a clear path of skill development to inspire a new generation of tech leaders. This summer our first cohort of students learned the coding languages that power the web (HTML, CSS and JavaScript) and worked together in small teams to build websites for local businesses.

During the 2015-2016 academic year CoderSpace students will learn how to build simple web applications with JavaScript and jQuery.
